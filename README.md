# Script for sorting files

This script is for sorting files in any given directory. It is a WIP, which means that not all file types are added yet. 
It works by looking at all the files in the given directory, it then prints out a list of all files and asks the user if they would like to sort the files.

If the user answers "yes" then the files will be sorted into the correct directories: .txt => texts, .jpg => images and so on. 
if there is no corresponding directory it will be created. If the file is not in the config.json file it will not be sorted.

If you'd like to add more file types just add them to the config.json file.

## How to use: 

Simply run the script, follow the instructions in the terminal. 

**Note: type in the ABSOLUTE and CORRECT path that you want sorted. With or without a trailing /, doesn't matter.**
